BubbleSeries Spec

Defines a bubble series in the chart.

Overview​

Compatible Specs​

BubbleSeries spec can be used multiple times along with any of the following specs:

Global Specs​

  • Settings
  • Tooltip
  • GroupBy
  • SmallMultiples

Specs​

  • Axis
  • AreaSeries
  • LineSeries
  • BarSeries
  • BubbleSeries
  • HistogramBarSeries
  • LineAnnotation
  • RectAnnotation

Overrides​

  • seriesType -> SeriesType.Bubble

Props​

All props can be found via LineSeriesProps type.

BasicSeriesSpec Props​

Inherits BasicSeriesSpec props

bubbleSeriesStyle​

  • Type: bubbleSeriesStyle?: RecursivePartial<BubbleSeriesStyle>

pointStyleAccessor​

  • Type: pointStyleAccessor?: PointStyleAccessor

An optional functional accessor to return custom color or style for point datum
