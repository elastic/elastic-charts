# `Spec`

Base spec for all chart types.

## Props {#props}

### `id`
- Type: <code>id: string</code>

unique Spec identifier

### `chartType`
- Type: <code>chartType: ChartType</code>

Chart type define the type of chart that use this spec

### `specType`
- Type: <code>specType: string</code>

The type of spec, can be series, axis, annotation, settings etc
