# `HistogramConfig`

Defines histogram config used for banded `LineSeries` and `AreaSeries`.

## Props {#props}

### `histogramModeAlignment`
- Type: <code>histogramModeAlignment?: [HistogramModeAlignment](/all-types/type-aliases/HistogramModeAlignment)</code>
- Default: `'start'`

Determines how points in the series will align to bands in histogram mode
