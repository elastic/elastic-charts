# `Postfixes`

Defines postfixes used for banded `BarSeries` or `AreaSeries`.

### `y0AccessorFormat`
- Type: <code>y0AccessorFormat?: string</code>
- Default: `upper`

Postfix for y0 accessor when using `y0Accessors`

### `y1AccessorFormat`
- Type: <code>y1AccessorFormat?: string</code>
- Default: `lower`

Postfix for y1 accessor when using `y1Accessors`
