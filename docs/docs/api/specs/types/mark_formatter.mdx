# `MarkFormatter`

Defines parameters for mark formatter.

:::info
Parameters only apply to `LineSeries` and `AreaSeries`
:::

## Props

### `markFormat`
- Type: <code>markFormat?: [TickFormatter](/all-types/type-aliases/TickFormatter)\<number\></code>

A function called to format every single mark value
