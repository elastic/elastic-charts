# `Goal` Spec

Defines a simple goal/guage/bullet chart.

:::danger
This spec has been deprecated and replaced by the [`BulletGraph`](./BulletGraph) spec. This will eventually be entirely removed from the library in future versions.
:::

## Overview {#overview}

## Compatible Specs

`Goal` is a standalone spec and cannot be combined with other specs except the `Settings` specs.

## Overrides

- `specType` -> `SpecType.Series`
- `chartType` -> `ChartType.Goal`
