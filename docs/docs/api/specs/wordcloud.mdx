# `Wordcloud` Spec

Defines a word cloud chart.

## Overview {#overview}

## Compatible Specs

`Wordcloud` is a standalone spec and cannot be combined with other specs except the `Settings` specs.

## Overrides

- `specType` -> `SpecType.Series`
- `chartType` -> `ChartType.Wordcloud`
