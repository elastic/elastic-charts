# `GroupBy` Spec

Defines grouping of `SmallMultiples`.

## Compatible Specs

`GroupBy` goes hand-in-hand with `SmallMultiples` spec to define a faceted chart.

Supported chart types include:

- All Cartesian/XY Charts
  - [`AreaSeries`](../xy/AreaSeries)
  - [`LineSeries`](../xy/LineSeries)
  - [`BarSeries`](../xy/BarSeries)
  - [`BubbleSeries`](../xy/BubbleSeries)
  - [`HistogramBarSeries`](../xy/HistogramBarSeries)
- [`Partition`](../Partition) Charts
- [`Heatmap`](../Heatmap) Charts

## Overrides

- `chartType` -> `ChartType.Global`
- `specType` -> `SpecType.IndexOrder`

### `by`

### `sort`
- Type: <code>sort: GroupBySort</code>

Sort predicate used to sort grouped data

### `format`
- Type: <code>format?: GroupByFormatter\<D\></code>

Formatter used on all `by` values.
