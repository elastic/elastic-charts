# `AreaSeries` Spec

Defines a area series in the chart.

## Overview {#overview}

## Compatible Specs

### `curve`
- Type: <code>curve?: CurveType</code>

The type of interpolator to be used to interpolate values between points

### `areaSeriesStyle`
- Type: <code>areaSeriesStyle?: RecursivePartial\<AreaSeriesStyle\></code>

### `stackMode`
- Type: <code>stackMode?: StackMode</code>

Stack each series using a specific mode: Percentage, Wiggle, Silhouette. The last two modes are generally used for stream graphs

### `pointStyleAccessor`
- Type: <code>pointStyleAccessor?: PointStyleAccessor</code>

An optional functional accessor to return custom color or style for point datum

### `fit`
- Type: <code>fit?: Exclude\<Fit, 'explicit'\> | FitConfig</code>

Fit config to fill `null` values in dataset
