## Props {#props}

All props can be found via [`RectAnnotationProps`](/all-types/type-aliases/RectAnnotationProps) type.

{/* TODO find a better way to represent these inherited props */}
### [`BaseAnnotationSpec`](../types/BaseAnnotationSpec) Props

Inherits [`BaseAnnotationSpec`](../types/BaseAnnotationSpec) props

### `outside`
- Type: <code>outside?: boolean</code>
- Default: `false`

Renders annotation outside of chart area within axis gutter

### `outsideDimension`
- Type: <code>outsideDimension?: number</code>

Dimension, either height or width, of outside annotation

### `zIndex`
- Type: <code>zIndex?: number</code>
- Default: `-1`

z-index of the annotation relative to other elements in the chart
