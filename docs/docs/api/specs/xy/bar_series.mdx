# `BarSeries` Spec

Defines a bar series in the chart.

## Overview {#overview}

### [`BasicSeriesSpec`](../types/BasicSeriesSpec) Props

Inherits [`BasicSeriesSpec`](../types/BasicSeriesSpec) props

### [`Postfixes`](../types/Postfixes) Props

Inherits [`Postfixes`](../types/Postfixes) props

### `enableHistogramMode`
- Type: <code>enableHistogramMode?: boolean</code>
- Default: `false`

### `barSeriesStyle`
- Type: <code>barSeriesStyle?: [RecursivePartial](/all-types/type-aliases/RecursivePartial)\<[BarSeriesStyle](/all-types/type-aliases/BarSeriesStyle)\></code>

Styles to apply to bar series. Overrides syles from [`Theme`](#gtd-theme)

### `stackMode`
- Type: <code>stackMode?: [StackMode](/all-types/type-aliases/StackMode)</code>
- Default: `'100%'`

Stack each series using a specific mode: Percentage, Wiggle, Silhouette.
The last two modes are generally used for stream graphs

### `styleAccessor`
- Type: <code>styleAccessor?: [BarStyleAccessor](/all-types/type-aliases/BarStyleAccessor)</code>
- Default: `'100%'`

Functional accessor to return custom color or style for bar datum

### `minBarHeight`
- Type: <code>minBarHeight?: number</code>
- Unit: `pixel`

Min height to render bars for highly variable data (i.e. ranges from 1 to 100,000).
