# `Heatmap` Spec

Defines a word cloud chart.

## Overview {#overview}

## Compatible Specs

`Heatmap` is a standalone spec and cannot be combined with other specs except for supported global specs.

:::warning
The `Heatmap` spec is **NOT** compatible with the [`Axis`](./xy/Axis) spec, it currently uses it's own internal axis implentation. The only axis controls are listed in the [`Props`](#props) below.
:::

## Overrides

- `specType` -> `SpecType.Series`
- `chartType` -> `ChartType.Heatmap`

## Props {#props}

All props can be found via [`HeatmapProps`](/all-types/type-aliases/HeatmapProps) type.

- Type: <code>colorScale: HeatmapBandsColorScale</code>

- Type: <code>xAccessor: Accessor\<D\> | AccessorFn\<D\></code>

### `yAccessor`
- Type: <code>yAccessor: Accessor\<D\> | AccessorFn\<D\></code>

- Type: <code>valueAccessor: Accessor\<never\> | AccessorFn</code>

- Type: <code>valueFormatter: ValueFormatter</code>
- Default: <code>(value) => `${value}`</code>

- Type: <code>xSortPredicate: Predicate</code>

- Type: <code>ySortPredicate: Predicate</code>
