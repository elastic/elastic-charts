# Troubleshooting `Chart`

If you've added your `@elastic/charts` to your app following the [consuming guide](./intro.mdx) and you are having issues rendering the chart see below...

## The `Chart` is not rendering

If you have a `Chart` defined on your page and you see an empty `div class="echContainer"></div>` element you are likely missing the css.

https://github.com/elastic/elastic-charts/wiki/Consuming-@elastic-charts#importing-css
