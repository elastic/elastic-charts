### Basic Bar Chart Prop Table


| Prop | Type | Default | Note |
|:------|:------:|:---------:|:------|
| id  `(required)` | `AxisId or string` | N/A | The id of the spec |
| name | `string` | N/A | The name or label of the spec |
| xScaleType `(required)`| `ScaleType (ScaleType.Ordinal | ScaleType.Linear | ScaleType.Time)` | ScaleType.Ordinal | The x axis scale type |
| yScaleType `(required)`| string[] scaleContinuousType | `ScaleType.Linear` | The y axis scale type |
| xAccessor `(required)`| Accessor | N/A | The field name of the x value on Datum object |
| yAccessors `(required)`| Accessor[] | N/A | An array of field names one per y metric value |
| data `(required)` | datum[] | N/A | An array of data |
